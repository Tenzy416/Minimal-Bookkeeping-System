# 大學生期末專題：極簡記帳系統 (Minimal Bookkeeping System)

這是一個專為大學生期末專題設計的**極簡記帳系統**，採用前後端分離的架構開發。系統內建響應式設計（Responsive Design），同時支援電腦與手機瀏覽器。

## 🌟 專案特點

- **前後端分離架構**：前端使用原生 HTML5、CSS3 及 JavaScript 打造高質感互動介面；後端使用 Flask 輕量級框架提供 RESTful API。
- **資料庫整合**：採用內建的 SQLite 資料庫（`expenses.db`），無須額外設定與安裝複雜的資料庫伺服器，即開即用。
- **跨來源資源共用 (CORS)**：後端 API 已完整配置 CORS 機制，方便前端透過不同連接埠或網域進行非同步（Asynchronous Fetch）資料對接。
- **直覺的可視化儀表板**：動態計算並展示「總收入」、「總支出」以及「目前結餘」，讓財務狀況一目了然。

## 📂 專案結構

```text
程式語言/
├── ExpenseForm/             # 記帳表單主要模組
│   ├── app.py               # Flask 後端 API 伺服器 (靜態檔案託管)
│   ├── index.html           # 前端主網頁介面

├── ExpenseTracker/          # 記帳歷史清單與儀表板模組
│   ├── app.py               # 備用後端 API 伺服器
│   ├── index.html           # 記帳清單前端介面
│   ├── styles.css           # 記帳清單樣式
│   └── app.js               # 記帳清單前端互動邏輯
└── .gitignore               # 排除暫存檔與資料庫上傳 (已配置)
```

## 🛠️ 執行與啟動步驟

### 1. 安裝必要套件
請確保您的電腦已安裝 Python 3，並於終端機執行以下指令安裝 Flask：
```bash
pip install Flask
```

### 2. 啟動後端伺服器
進入 `ExpenseForm` 資料夾，並執行 `app.py`：
```bash
cd ExpenseForm
python app.py
```

伺服器啟動後，控制台會顯示以下訊息：
```text
==================================================
[啟動中] 大學生期末專題：極簡記帳系統 API 伺服器啟動中...
[網址] 請在瀏覽器打開此網址進行操作: http://127.0.0.1:5000/
==================================================
```

### 3. 開啟網頁
在瀏覽器中輸入 `http://127.0.0.1:5000/`，即可直接開始使用精美的記帳系統！
